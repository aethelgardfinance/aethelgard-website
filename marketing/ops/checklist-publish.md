# Pre-Publish Checklist

Every draft passes through this gate before going out. No exceptions. If a check fails, the draft goes back, not out.

---

## 1. Spine + pivot
- [ ] The piece ladders up to the spine sentence in `brain/positioning.md`
- [ ] Exactly ONE of the three pivots (A founders / B family / C privacy) is chosen — not blended
- [ ] The pillar (1/2/3/4) is identified and matches `plan/content-pillars.md` rotation rules

## 2. Voice
- [ ] Read against `brain/voice.md` rules
- [ ] No banned phrases (`ops/style-guide.md`)
- [ ] At least one specific, concrete detail (number, place, moment, phrase)
- [ ] Em-dash density ≤ 1 per paragraph average
- [ ] No "AI tells" (it's not just X it's Y; in the age of; whether you're a / a / a)
- [ ] Sentence rhythm varies; not all medium-length
- [ ] **Dovedi check passes** — no scarcity countdowns, no "no fluff/no hype" disclaimers, no fear-of-loss framing, no yield/profit promises, no self-anointed rankings, no guilt-trip unsubscribe, no "valuable" self-description, no headline-case-everything, no venue-as-credibility, no commercial-copy emojis
- [ ] **Cramer check passes** — no catchphrases, no buy/sell/hold language, no bullish/bearish market positioning, no prediction reversals, no exclamation marks in body copy, no theatrical urgency, no fandom framing, no network-TV loud-fast register

## 3. Truth + confidentiality
- [ ] Nothing claims credentials Cherie does not hold (see `brain/credentials.md`)
- [ ] No client identifier is reverse-engineerable
- [ ] No regulated financial advice ("you should X")
- [ ] No HMRC speculation
- [ ] Numbers are accurate; figures cited are sourced

## 4. Mechanics
- [ ] British English
- [ ] GBP currency formatting
- [ ] Spell-checked
- [ ] Read aloud once
- [ ] Internal links work
- [ ] (Blog only) target keyword in H1 and first 100 words
- [ ] (Blog only) meta description 140–155 chars
- [ ] (Blog only) all images have alt text
- [ ] (LinkedIn only) hook line works as standalone preview
- [ ] (LinkedIn only) under 1300 chars unless deliberately long-form

## 5. Action
- [ ] Closes with a clear next step OR a deliberate silence (not a generic "what do you think?")
- [ ] If newsletter: one CTA (sign up, download, apply for design partner) — not three
- [ ] If LinkedIn: at most one link in comments, never in the post body (LinkedIn algorithm penalty)

## 6. Final two questions

- [ ] **Would I be proud to have this on the About page?** If no, rewrite.
- [ ] **Does this sound like Cherie or like a marketing department?** If marketing, rewrite.

---

## Failed checks

If any check fails, the draft is returned to `drafts/` with `[BLOCKED: <reason>]` at the top. Cherie sees blockers before she edits — saves time.
